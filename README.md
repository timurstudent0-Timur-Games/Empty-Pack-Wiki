## 🛠 Як використовувати цей пак?
1. Завантажте останню версію з [Modrinth](https://modrinth.com/resourcepack/start_pack).
2. Помістіть `.zip` файл у папку `resourcepacks` вашого Minecraft.
3. У грі виберіть цей пак. Тепер у вас є готова структура папок для створення власних текстур!
